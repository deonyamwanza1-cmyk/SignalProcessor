Digital Signal Processing: C++ Low-Pass Filter
Connecting Electrical Engineering and Software Logic
The project implements the equivalent of a physical RC Low-Pass Filter. It processes noisy data simulating noisy signals.

Processes:
Signal Generation: A Python-based simulation engine that creates sine waves with Additive White Gaussian Noise (AWGN).

C++ DSP Engine: An optimized Infinite Impulse Response (IIR) filter class.

Visual Validation: Automated plotting using Matplotlib to verify noise reduction.
